Documentacion de ejercicios - Estructuras condicionales

En esta parte del proyecto me enfoque en poner en practica lo aprendido sobre el control de flujo en Python. Para el primer archivo, que es el ejercicio basico, trabaje una logica simple de banco donde se compara el saldo con un retiro usando if y else para decidir si la transaccion se aprueba o no. Es un ejemplo sencillo para entender como Python toma decisiones binarias.

Para el ejercicio multiple decidi aplicar un sistema de calificaciones. Aqui use la estructura if-elif-else para evaluar diferentes rangos de notas, lo que permite que el programa maneje varias opciones en orden hasta encontrar la correcta. Me sirvio para entender que el orden de las condiciones es muy importante para que el codigo no se equivoque al clasificar.

Finalmente, en el ejercicio avanzado implemente el match-case que es algo mas nuevo en Python. Lo use para crear un selector de frutas donde se compara lo que escribe el usuario con diferentes patrones. Tambien agregue el uso de un comodin para los casos donde la fruta no se reconoce, lo que hace que el programa sea mas robusto ante errores del usuario.
